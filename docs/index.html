Joey's Web Crypto Library is a wrapper around the browser based Javascript Web Crypto API and the Node Crypto Module . JWCL provides a high level interface, secure defaults, and network portable output. JWCL can be used interchangeably between the browser and server.

JWCL is easy to get started with.

To encrypt run

const encryptedMessage = await jwcl.encrypt('secret', 'message');

and to decrypt

await jwcl.decrypt('secret', encryptedMessage); // returns 'message'
